# 🧬 Cancer Gene Expression Analysis

A comprehensive computational analysis of cancer-related gene expression patterns. This project demonstrates fundamental principles of bioinformatics and precision oncology through exploratory data analysis and visualization.

## 📋 Project Overview

This project analyzes the expression patterns of six key oncology-related genes:

- **TP53** - Tumor suppressor protein (p53) - "Guardian of the Genome"
- **BRCA1/BRCA2** - Breast cancer susceptibility genes
- **EGFR** - Epidermal growth factor receptor
- **MYC** - Myelocytomatosis oncogene
- **PTEN** - Phosphatase and tensin homolog

### Key Features

✨ **Educational Focus**: Designed for beginners in bioinformatics and computational biology  
🔬 **Professional Structure**: Academic-grade organization suitable for portfolio and research contexts  
📊 **Data Analysis**: Statistical summary and comparative gene expression analysis  
📈 **Visualization**: Professional, publication-quality graphics  
💡 **Interpretation**: Biological context linking expression patterns to cancer phenotypes

## 🚀 Getting Started

### Prerequisites

- Python 3.6 or higher
- Jupyter Notebook or JupyterLab
- Required packages: pandas, matplotlib, numpy

### Installation

1. Clone the repository:
```bash
git clone https://github.com/SabrinaMoraes22/cancer-analysis.git
cd cancer-analysis
```

2. Create a virtual environment (recommended):
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install jupyter pandas matplotlib numpy
```

4. Launch Jupyter Notebook:
```bash
jupyter notebook
```

5. Open the notebook file: `Cópia_de_cancer_gene_expression_analysis.ipynb`

## 📈 Key Findings

The analysis reveals a gene expression profile characteristic of aggressive malignancies:

- **High Expression Drivers**: TP53 (8.5), EGFR (7.2), MYC (6.8)
- **Low Expression Suppressors**: BRCA1 (2.1), BRCA2 (3.0), PTEN (4.5)

This pattern suggests potential therapeutic opportunities including EGFR inhibition, MYC targeting, and synthetic lethality approaches.
